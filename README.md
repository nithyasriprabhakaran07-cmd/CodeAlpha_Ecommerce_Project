# CodeAlpha E-Commerce Project

## Overview

A Django-based E-Commerce website developed as part of the CodeAlpha Internship.

## Features

* User Registration & Login
* Product Listing
* Product Search
* Product Details Page
* Shopping Cart
* Wishlist
* Checkout System
* Order Processing
* Responsive Bootstrap UI

## Technologies Used

* Python
* Django
* SQLite
* HTML
* CSS
* Bootstrap
